# Quality Skill Routing

Use this category for tests, smoke checks, releases, and regression evidence.

## Route By Intent

- `regression-suite`: use to design or run regression checks for changed behavior.
- `smoke-check`: use for fast confidence that an edited path starts, responds, or compiles.
- `test-setup`: use to configure deterministic, isolated, reversible test environments.
- `test-helpers`: use when adding or refactoring helper utilities for tests.
- `release-checklist`: use before packaging or publishing.

## Evidence Rule

Tests are supporting evidence. User-facing behavior still needs a real surface check when the workflow claims completion.
