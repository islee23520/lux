# Studio Skill Routing

Use this category for game-studio collaboration workflows that are not direct engine operations.

## Route By Intent

- `studio-brainstorm`: use for game concept ideation from vague ideas to a structured concept brief.
- `studio-sprint-plan`: use to create, update, or summarize sprint plans from milestones, capacity, risks, and ready work.
- `studio-gate-check`: use to decide whether a project is ready to advance to the next phase.
- `studio-help`: use for fast workflow orientation and the single best next action.

## Do Not Use For

- Code review. Use `../review/studio-code-review` or `../review/code-review`.
- LUX runtime commands. Use `../workflow/AGENTS.md`.
